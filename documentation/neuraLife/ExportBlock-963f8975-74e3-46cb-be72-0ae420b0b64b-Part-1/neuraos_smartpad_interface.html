4.2 Solution by Factorisation
A quadratic equation ax² + bx + c = 0 can be solved by expressing bx as a sum of two terms whose product equals ac.
Standard form decomposition
ax² + bx + c = 0
Find p, q: p + q = b · p × q = ac
(x + p/a)(x + q/a) = 0
Figure 4.3 — Factorisation method
This works when b² − 4ac is a perfect square. Otherwise use the quadratic formula.
